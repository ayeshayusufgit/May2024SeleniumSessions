HasAuthentication is an interface implemented in the ChromiumDriver class
ChromiumDriver is extending the RemoteWebDriver and implementing the register() of the HasAuthentication Interface
This interface is only implemented in ChromiumDriver only(only is Edge n Chrome based browser)
